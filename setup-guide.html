Enterprise Email System - Database Migration

⚠️ Action Required: The database tables need to be created before the email system can function.

Step 1: Run Database Migration PENDING

Click the button below to create the required database tables:

🔧 Run Migration Script

This will create the following tables:

  • form_submissions - Stores all form data
  • rate_limit_log - Tracks rate limiting

Step 2: Configure SMTP Settings

Update your .env file with production SMTP credentials:

SMTP_USERNAME=your-email@gmail.com SMTP_PASSWORD=your-app-password ADMIN_EMAIL=admin@wavelengthenterprises.com

📌 Gmail Users: Generate an App Password from Google Account settings

Step 3: Delete Migration File

After successful migration, delete the migration file for security:

Delete: config/run_migration.php

Step 4: Test Forms

Test the email system with the example forms:

  • Contact Form
  • Quote Form
